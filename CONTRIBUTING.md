# Contributing

Thank you for considering contributing!

## Build Requirements

- C++11 compatible compiler (GCC 4.8+ / MSVC 2015+ / Clang 3.3+)
- Make (optional)

## Build & Run

`ash
g++ -std=c++11 -o shelfstack src/*.cpp -Iinclude
./shelfstack
`

## Code Style

- Use 4-space indentation
- Follow C++11 best practices
- Prefer STL containers over raw arrays
